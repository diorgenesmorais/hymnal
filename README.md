# Hymnal - tocador de hinos

- Após empacotar para um executável, salve os hinos nesta extrutura

```text
dist/
├── public/
│   └── videos/     # Coloque seus vídeos aqui
│       ├── video1.mp4
│       ├── video2.mp4
│       └── ...
├── hymnal.exe
```

- Comando para criar o executável

```bash
pkg . --targets windows
```
